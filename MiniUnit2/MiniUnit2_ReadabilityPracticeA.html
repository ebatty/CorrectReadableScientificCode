Brainstorming Exercise: What makes code less readable?#

Please note that the above doodle is not CC-BY and we’d need to replace it. I just wanted to visualize what the unit would look like with cute doodles

Please look at each of the following code snippet. Without running the code, try to brainstorm:

  • Any thing that help you understand what the code is doing

  • Any thing that decrease the readability of the code. What parts of the code are unclear or take you a little extra time to understand?

If completing this with others, please take a few minutes to brainstorm by yourselves and then discuss.

not_tall_enough = rdrhgt < 5
